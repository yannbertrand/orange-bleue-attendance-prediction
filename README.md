# orange-bleue-attendance-prediction

**Stability: 1 - Experimental**

L'application mobile des salles de sport L'Orange Bleue permet de suivre le nombre de personnes présentes en direct.

<img width="147" height="213" alt="IMG_6992" src="https://github.com/user-attachments/assets/5b94fa9e-56c0-4aa6-861b-af9e65f5216e" />


Ce projet récupère ces stats régulièrement pour les stocker dans des fichiers : 
- [`attendance.csv`](https://singular-centaur-177656.netlify.app/attendance.csv) qui conserve les visites et estime l'évolution au cours du temps,
- [`prediction.csv`](https://singular-centaur-177656.netlify.app/prediction.csv) qui stocke les informations des cours à venir pour déduire la présence future.

Ces données sont ensuite utilisées à des fins statistiques pour éviter les grosses fréquentations (usage purement personnel).
